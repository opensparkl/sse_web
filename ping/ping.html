  • Loading this page over https?
    • Some browsers including Safari prevent you connecting to an sse over http.
    • Connect to the sse using https instead (see below).
  • Connecting to an sse over https whose server certificate is self-signed?
    • Load https://[sse.domain][:port]/sse/ping directly in your browser first, and click past the security prompts.
    • Or, retrieve the sse server certificate .pem file and install it in your operating system or browser as a trusted cert.
